# Images Folder

Place your background image here.

## Recommended:
- **Filename:** `background.jpg` (or update the path in `index.html`)
- **File size:** Under 1MB for fast loading
- **Dimensions:** 1920x1080 or higher for high-resolution displays
- **Formats supported:** JPG, PNG, GIF, WebP

## Where to find free Christmas images:
- [Unsplash](https://unsplash.com/s/photos/christmas)
- [Pexels](https://www.pexels.com/search/christmas/)
- [Pixabay](https://pixabay.com/images/search/christmas/)

## Tips:
- Use landscape orientation for best results
- Ensure image is festive and appropriate for recipients
- Compress large images using tools like [TinyPNG](https://tinypng.com/)
