# Logo Files

This folder contains the logo that appears in the top left corner of the Christmas card.

## Required File

Place your logo image here:
- **File name:** `hciuse.png`
- **Recommended formats:** PNG (with transparency), JPG, SVG
- **Recommended size:** 200x200 pixels to 400x400 pixels
- **File size:** Keep under 100KB for fast loading

## How to Add Your Logo

1. Save your logo image as `hciuse.png` in this folder
2. The logo will automatically appear in the top left corner of the card
3. The logo has a white semi-transparent background with rounded corners
4. On mobile devices, the logo automatically scales down

## Styling

The logo is styled with:
- Fixed position in top left corner
- White background with transparency
- Rounded corners and subtle shadow
- Hover effect (slight lift)
- Responsive sizing for mobile devices

## Customization

To customize the logo styling, edit the `.logo-container` and `.site-logo` CSS classes in `styles.css`.
